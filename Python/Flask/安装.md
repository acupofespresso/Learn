### 创建虚拟环境
* Windows
```
# 在当前目录创建venv文件
py -3 -m venv venv
# 激活虚拟环境
venv\Scripts\activate
```
* Linux
```
# 创建项目文件夹
$ mkdir myproject
$ cd myproject
# 在项目文件夹下创建虚拟环境
$ python3 -m venv venv
# 激活
$ . venv/bin/activate
```
